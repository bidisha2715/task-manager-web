# Task Manager Web App

A responsive Task Manager web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add and delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Search functionality
- Due dates for tasks
- Dark mode toggle
- Task counter
- Data persistence using localStorage

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## 📂 Project Structure
- index.html
- style.css
- script.js

## 💡 Description
This project demonstrates DOM manipulation, event handling, and localStorage usage to build a fully functional client-side task management application.
